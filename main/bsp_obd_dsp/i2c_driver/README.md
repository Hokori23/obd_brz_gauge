# `i2c_driver`

这是 I2C 总线封装层。

在这个项目里，I2C 主要服务于：

- `TCA9554` IO 扩展芯片
- `CST816` 触摸芯片
- 其他挂在 I2C 上的外设

## 你需要知道的背景

I2C 是一条两线总线：

- `SCL`：时钟
- `SDA`：数据

多个设备可以挂在同一条总线上，通过地址区分。

## 这个模块的职责

- 初始化 I2C 控制器
- 提供基础读写函数
- 给更上层驱动复用
