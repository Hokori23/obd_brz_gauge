# `exio`

`exio` 是 external IO 的缩写。这里封装的是 `TCA9554` IO 扩展芯片。

## 为什么需要它

ESP32 自带 GPIO 有限，而开发板上的屏幕复位、背光控制、其他控制脚可能需要额外 IO。

所以项目通过 I2C 挂了一颗 `TCA9554`，把少量控制引脚扩出来。

## 模块职责

- 初始化 `TCA9554`
- 配置引脚方向
- 输出控制电平
- 给 LCD/其他外设提供复位或控制信号

## 在项目里的位置

启动顺序里会先 `I2C_Init()`，再 `EXIO_Init()`，之后 LCD 初始化会依赖它。

## 你读代码时关注什么

- 哪些位对应哪些外设控制脚
- 初始化时哪些引脚被拉高/拉低
- 屏幕复位流程是否经过这里
