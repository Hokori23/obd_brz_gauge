# `CST816.c/.h` 阅读说明

这是触摸控制器 `CST816` 的具体驱动。

## 主流程

1. 创建 `esp_lcd_touch_handle_t`
2. 配置中断脚和复位脚
3. 复位触摸芯片
4. 读取芯片 ID
5. 暴露 `read_data/get_xy` 回调给 LVGL

## 你该重点看哪几段

### `esp_lcd_touch_new_i2c_cst816()`

这是构造函数，负责把设备句柄和操作函数组装起来。

### `read_data()`

从触摸寄存器读取点位数据，写入内部缓存。

### `get_xy()`

把缓存里的点坐标提供给 LVGL 输入系统。

### `reset()`

这里不是直接操作 ESP32 GPIO，而是通过 `Set_EXIO()` 走 IO 扩展芯片完成复位。

这说明：

- 触摸驱动依赖 IO 扩展
- 不是所有控制脚都直接挂在 ESP32 上

## 你可以这样理解

这个文件把“一个 I2C 触摸芯片”适配成了 LVGL 能消费的输入设备。
