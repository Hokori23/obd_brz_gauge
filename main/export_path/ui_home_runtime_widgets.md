# `ui_home_runtime_widgets` 模块说明

这是首页卡片构建模块。
核心职责：指标卡片创建、槽位装配、页面布局挂接、局部刷新辅助。

## 模块负责什么

- 按页面配置创建首页卡片和指标控件
- 处理槽位数量、顺序和布局挂载
- 把业务侧指标映射到具体 UI 控件
- 为首页运行时刷新提供控件级入口

## 为什么单独拆出来

首页运行时关心的是“什么时候刷新”和“当前该显示哪一页”。
这个模块关心的是“这一页里有哪些卡片”和“卡片该怎么摆”。
把两者拆开后，布局问题和状态问题不会互相缠在一起。

## 维护注意点

- 新增卡片类型时，优先在这里补齐创建和挂载逻辑
- 如果只是刷新时机变化，优先回到 `ui_home_runtime.c` 处理
- 圆屏安全区和尺寸适配应继续复用 `ui_platform` / `ui_round_shell`
