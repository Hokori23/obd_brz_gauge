# `png_to_lvgl.py` 阅读说明

这个脚本负责把 PNG 转成 LVGL 可直接编译进固件的 C 资源文件。

## 它做了什么

1. 读取 `tools/sklogo.png`
2. 缩放到目标尺寸
3. 把透明 PNG 先铺到黑底上
4. 把 RGB888 转成 RGB565
5. 生成 `lv_img_dsc_t` 对应的 C 文件

## 为什么要转成 C 数组

因为在这类固件项目里，图片资源常常不是运行时从文件系统加载，而是：

- 直接编译进固件
- 作为 `const uint8_t[]` 常量存在 flash 里

## 你可以怎么理解

它相当于前端打包阶段的“图片预处理脚本”，只是目标格式不是 Web 图片资源，而是 LVGL 的 C 资源描述。
